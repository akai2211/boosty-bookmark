# Память агента: Boosty Bookmark

В этом файле зафиксированы все ключевые правила, предпочтения и технические детали проекта для совместной работы.

---

## 1. Языковые настройки и стиль общения
*   **Язык общения:** Строго русский язык (в ответах, комментариях в коде и сообщениях коммитов).
*   **Стиль принятия решений:** Никогда не спешить. Всегда запрашивать явное согласие пользователя перед важными изменениями (переименования, удаление, рефакторинг). Предлагать альтернативные варианты.

---

## 2. Правила работы с Git и тестированием (КРИТИЧЕСКИ ВАЖНО)
*   **Запрет на автоматический push:** **НИКОГДА** не выполнять команду `git push` или любые другие команды, отправляющие изменения в удаленный репозиторий, автоматически. Это действие выполняется **только** после прямого указания Акая (*«сделай пуш»*, *«запушь»*).
*   **Запрет на автоматический коммит:** **НИКОГДА** не делать локальные коммиты (`git commit`) без прямого указания Акая (*«закоммить»*, *«сделай коммит»*).
*   **Правило запуска тестов:** Запуск тестов (`npm test`, `playwright`, `vitest` и др.) разрешен **только перед коммитом** (после получения команды закоммитить) для валидации изменений. В других случаях запускать тесты запрещено.
*   **Правило обновления версии перед коммитом:** Перед совершением любого локального коммита ИИ-агент обязан убедиться в актуальности версии в `manifest.json` и запустить скрипт сборки (`npm run build`), чтобы автоматически синхронизировать версию в `manifest.firefox.json` и `package.json`. Запрещается коммитить изменения с устаревшей версией. Устанавливать мажорные версии (например, `1.0.0`, `2.0.0`) имеет право **только Акай**. Агенты могут использовать только минорные и патч-версии в рамках текущей ветки (например, `0.9.1`, `0.9.9.1`, `0.9.162` — максимум до 4 чисел, разделенных точками, согласно лимитам Chrome/Firefox манифестов).
*   **Информационные действия разрешены:** Выполнение вспомогательных команд `git add`, `git status`, `git diff`, `git log` разрешено без согласования.
*   **Сообщения коммитов:** Все сообщения коммитов пишутся исключительно на русском языке.
