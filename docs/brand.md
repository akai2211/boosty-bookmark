## 3. Подробная матрица донатов и монетизации

Система поддержки полностью добровольная. Разделена на регулярную (подписки) и разовую (фиат / крипта):

| Тип поддержки | Платформа / Инструмент | Роль в проекте |
| :--- | :--- | :--- |
| **Регулярная** | **Boosty (Основная)** | Ежемесячная подписка (100–200 ₽/мес) по желанию пользователя. Здесь же публикуются новости. |
| **Разовая (Фиат)** | **YooMoney** | Прямой перевод внутри РФ. |
| **Разовая (Фиат)** | **DonationAlerts** | Альтернативный вариант для рублевых карт. |
| **Криптовалюта** | **TON** | Прямой перевод на кошелек (удобно для аудитории из Telegram). |
| **Криптовалюта** | **USDT (TRC-20)** | Стабильный универсальный вариант для обхода ограничений. |

## 5. Логика связей в интерфейсе расширения

Когда пользователь открывает вкладку «О расширении», ссылки ведут строго по архитектурным точкам:
- **«GitHub Репозиторий»** → Ссылка на open-source репозиторий проекта.
- **«Связаться с автором / Предложить идею»** → Перенаправление в Telegram (канал или чат).
- **«Не является официальным расширением Boosty»** → Обязательный нейтральный дисклеймер.
- **Блок «ПОДДЕРЖАТЬ ПРОЕКТ»** (Кнопки):
  1. **Boosty** (Первая, выделенная кнопка)
  2. **DonationAlerts**
  3. **ЮMoney**
  4. **Крипта (TON / USDT)**

## 6. Важные правила и решения

- **Никакой «официальности»:** Четко пишем, что расширение независимое, чтобы не ловить баны от платформы. Работаем только с DOM-деревом сайта.
- **Единый новостной центр:** Чтобы не размазывать аудиторию, все посты о разработке и апдейтах выходят на Boosty. Telegram используется как инструмент быстрой связи и дублирования.
- **Email для модераторов:** Для регистрации в Web Store используем временную почту-заглушку (Gmail/Yandex). Полноценную почту на домене поднимем позже, когда дойдут руки до сайта.
