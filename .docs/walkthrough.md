# Результаты разработки: Boosty Bookmark

Мы успешно создали и протестировали Chrome-расширение, помогающее отслеживать прогресс просмотра видеороликов и озвучек манхвы и маньхуа на странице автора `lightfoxmanga` на Boosty.

Все файлы расширения находятся в вашей рабочей директории: [Boosty Bookmark](file:///home/akai/projects/boosty-bookmark).

## Архитектура и созданные файлы

1.  **[manifest.json](file:///home/akai/projects/boosty-bookmark/manifest.json)**
    Конфигурационный файл Manifest V3. Настроен на запуск расширения только на страницах `https://boosty.to/lightfoxmanga*`. Запрашивает разрешения `storage` и `unlimitedStorage` для надежного локального хранения базы постов и прогресса.
2.  **[content.js](file:///home/akai/projects/boosty-bookmark/content.js)**
    Основной скрипт расширения. Он отвечает за:
    *   Первичную фоновую загрузку всех 1250+ постов с API Boosty и их кэширование.
    *   Фоновую быструю синхронизацию новых глав и актуализации лайков (загружает только первую страницу при открытии).
    *   Группировку постов по тайтлам (на основе чистых тегов) и автоматическую сортировку по тирам подписок.
    *   Синхронизацию прогресса через лайки (`isLiked`) и ручные чекбоксы.
    *   Автосохраняемый блокнот для заметок по каждому произведению.
    *   Управление динамическим интерфейсом.
3.  **[styles.css](file:///home/akai/projects/boosty-bookmark/styles.css)**
    Стили интерфейса. Наследуют шрифты и нативно подстраиваются под темный или светлый фон Boosty (поддержка темной темы сайта). Используют эффект матового стекла (glassmorphism) и плавные переходы.
4.  **[icons/](file:///home/akai/projects/boosty-bookmark/icons/)**
    Папка с PNG-иконками расширения трех обязательных размеров (16x16, 48x48, 128x128), сгенерированными скриптом **[generate_icons.py](file:///home/akai/projects/boosty-bookmark/generate_icons.py)**.

## Инструкция по установке в Google Chrome

Чтобы запустить и протестировать расширение, выполните следующие простые шаги в вашем браузере Chrome:

1.  Откройте браузер Chrome и перейдите по адресу **`chrome://extensions/`** (введите это в адресную строку).
2.  В правом верхнем углу страницы включите переключатель **«Режим разработчика»** (Developer mode).
3.  В левом верхнем углу нажмите появившуюся кнопку **«Загрузить распакованное расширение»** (Load unpacked).
4.  В открывшемся окне выбора папки выберите директорию проекта:
    `/home/akai/projects/boosty-bookmark`
5.  Расширение успешно загрузится и появится в списке.
6.  Перейдите на страницу **`https://boosty.to/lightfoxmanga`**. Справа по центру экрана появится оранжевая плавающая кнопка-триггер.

## Ключевые возможности, готовые к тестированию

*   **Первичная загрузка:** При первом клике на оранжевую кнопку расширение покажет шкалу загрузки и за пару секунд загрузит всю историю постов (1250+ постов), сгруппировав их в 190+ уникальных тайтлов.
*   **Автоматическое распределение по тирам:** Тайтлы сами разделятся по группам доступа (`Любителям ютуба`, `Любителям манги`, `Любителям пика`, `Лисямбы мои`, `Для шейхов`, `Только для девушек`).
*   **Цветовые индикаторы:**
    *   🟡 *Желтый круг* — есть новые или еще непросмотренные главы.
    *   🟢 *Зеленый круг* — все доступные главы тайтла просмотрены.
    *   ⚪ *Серый круг* — просмотр тайтла еще не начато.
    *   🔴 *Красный круг* — тайтл перенесен в «Брошено» (скрывается из других списков).
*   **Синхронизация через лайки (автоматическая):** Если вы ставите лайк посту на Boosty, расширение при фоновом обновлении (или при ручной синхронизации) автоматически пометит эту главу как просмотренную (зеленая галочка). Это идеальный способ переносить прогресс между устройствами!
*   **Индивидуальные блокноты:** Нажмите на любой тайтл, откроется детальный вид. Напишите заметку в текстовом поле — она мгновенно сохранится локально.
*   **Автоотметка при открытии:** Возможность автоматически отмечать главу как просмотренную при клике по её ссылке в списке (по умолчанию эта функция выключена, но её можно активировать в настройках сайдбара).
*   **Ручное сворачивание панели:** Закрытие (сворачивание) панели происходит только по клику на кнопку-стрелочку в правом верхнем углу (автозакрытие при клике снаружи отключено по просьбе пользователя для предотвращения случайного закрытия).
*   **Динамическая SPA-маршрутизация:** Расширение отслеживает переходы по страницам внутри Boosty. При переходе к другим блогерам кнопка и нав-панель скрываются, а при возвращении на страницу `lightfoxmanga` — снова отображаются.
*   **Кнопка «Настройки» (шестеренка в шапке):** Открывает панель настроек, где объединены:
    *   **Экспорт и импорт прогресса:** скачивание и загрузка JSON-файла резервной копии.
    *   **Управление поведением:** чекбоксы для включения/выключения отслеживания по лайкам и автоматической отметки глав при переходе по ссылке.
    *   **Порядок вкладок:** возможность настраивать расположение вкладок в боковой панели. Раздел сделан сворачиваемым (по умолчанию скрыт, раскрывается по клику на заголовок) для экономии пространства. Поддерживается как перетаскивание мыши (Drag & Drop) за иконку с точками слева от названия, так и традиционное перемещение кнопками стрелок ▲ и ▼.
    *   **Масштаб интерфейса:** выпадающий список для переключения масштаба (zoom) всей боковой панели (100%, 110%, 120%, 125%, 130%, 140%, 150%) для комфортного чтения и кастомизации под разрешение экрана.
