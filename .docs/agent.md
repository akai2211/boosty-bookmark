# Память агента: LightFox Boosty Bookmark

В этом файле зафиксированы все ключевые правила, предпочтения и технические детали проекта для совместной работы.

---

## 1. Языковые настройки и стиль общения
*   **Язык общения:** Строго русский язык (в ответах, комментариях в коде и сообщениях коммитов).
*   **Стиль принятия решений:** Никогда не спешить. Всегда запрашивать явное согласие пользователя перед важными изменениями (переименования, удаление, рефакторинг). Предлагать альтернативные варианты.

---

## 2. Ограничения на использование Git (КРИТИЧЕСКИ ВАЖНО)
*   **Запрет на автоматические коммиты:** **НИКОГДА** не выполнять команды `git add`, `git commit`, `git push` или любые другие модифицирующие репозиторий команды Git автоматически. 
*   **Согласование коммитов:** Каждое действие с Git (добавление в индекс, фиксация изменений) должно выполняться **только** после того, как пользователь напрямую даст команду: *«Закоммить»*, *«Сделай коммит»* или *«Добавь в гит»*.
*   **Сообщения коммитов:** Все сообщения коммитов пишутся исключительно на русском языке.

---

## 5. Технические особенности расширения
*   **Метод отслеживания глав:** Реализован через лайки Boosty (`isLiked`). При фоновом обновлении лайкнутые посты автоматически помечаются как просмотренные.
*   **Автономность разработки:** Доступ к закрытому платному контенту не требуется для разработки, так как метаданные заголовков глав и теги авторов получаются через публичное API Boosty.
*   **SPA-маршрутизация и видимость:** Сайт Boosty функционирует как SPA. Расширение загружается на всех страницах `https://boosty.to/*`, динамически скрывая/показывая интерфейс при изменении URL (через перехват истории переходов и резервный таймер) в зависимости от того, находится ли пользователь на странице автора `lightfoxmanga`.
*   **Автозакрытие панели:** Реализовано закрытие боковой панели при клике в любую область страницы за пределами самой панели и кнопки её вызова.
