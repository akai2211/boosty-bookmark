# Память агента: Boosty Bookmark

В этом файле зафиксированы все ключевые правила, предпочтения и технические детали проекта для совместной работы.

---

## 1. Языковые настройки и стиль общения
*   **Язык общения:** Строго русский язык (в ответах, комментариях в коде и сообщениях коммитов).
*   **Стиль принятия решений:** Никогда не спешить. Всегда запрашивать явное согласие пользователя перед важными изменениями (переименования, удаление, рефакторинг). Предлагать альтернативные варианты.

---

## 2. Правила работы с Git (КРИТИЧЕСКИ ВАЖНО)
*   **Запрет на автоматический push:** **НИКОГДА** не выполнять команду `git push` или любые другие команды, отправляющие изменения в удаленный репозиторий, автоматически. Это действие выполняется **только** после прямого указания Акая (*«сделай пуш»*, *«запушь»*).
*   **Локальные действия разрешены:** Выполнение команд `git add`, `git commit`, `git status`, `git diff` и других локальных операций с репозиторием разрешено без предварительного согласования.
*   **Сообщения коммитов:** Все сообщения коммитов пишутся исключительно на русском языке.
