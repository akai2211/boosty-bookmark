## Улучшение реактивности (живости) расширения

- [x] **А. Перехват fetch — мгновенное отображение лайков с Boosty в расширении**
    - Патчим `window.fetch`, чтобы перехватывать POST-запросы к `/reaction` (лайк на сайте Boosty)
    - При перехвате успешного ответа — мгновенно обновляем `isLiked` в кэше и перерисовываем чекбокс в расширении
    - Без необходимости синхронизации, задержка = 0
- [x] **Б. Умная инкрементальная синхронизация (замена полной 25-секундной загрузки)**
    - Вместо загрузки всех 1250+ постов, загружаем страницы API до тех пор, пока не встретим N подряд известных постов с неизменённым `isLiked`
    - Для обычного обновления это ~1-2 страницы (2-3 секунды вместо 25)
    - Полная синхронизация остаётся доступной как принудительная опция
- [x] **В. DOM-синхронизация лайка обратно на страницу Boosty**
    - При постановке галочки в расширении — вместо (или помимо) API-запроса находим и кликаем настоящую кнопку лайка на странице Boosty программно
    - Тогда и сайт обновится визуально, и расширение — без перезагрузки F5
