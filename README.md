# Boosty Bookmark 🔖

**Boosty Bookmark** — это удобное браузерное расширение для отслеживания прогресса чтения манхвы, маньхуа и просмотра видео на платформе **Boosty**. 

Расширение помогает структурировать публикации автора по отдельным произведениям (тайтлам), отмечать просмотренные главы, сохранять личные заметки и автоматически синхронизировать прогресс между вашими устройствами.

## ✨ Основные возможности

* **Удобная боковая панель:** Показывает все ваши произведения на одной вкладке прямо на странице Boosty.
* **Категоризация тайтлов:** Сортировка произведений по спискам: *«Избранное»*, *«Смотрю»*, *«Завершено»*, *«Брошено»* и *«Не отслеживаю»*.
* **Умное отслеживание глав:** Автоматическое или ручное отмечание просмотренных глав.
* **Синхронизация по лайкам:** Возможность автоматически отмечать главы как просмотренные, если вы поставили лайк соответствующему посту на Boosty.
* **Облачная синхронизация (WebDav):** Безопасный бэкап и автоматическая синхронизация прогресса с вашим облачным хранилищем (Яндекс.Диск, Nextcloud, NAS и др.) в формате ZIP-архива.
* **Запоминание прогресса плеера:** Автоматическое сохранение и восстановление времени воспроизведения встроенных видеоплееров Boosty, а также принудительный выбор предпочитаемого качества видео.
* **Встроенный блокнот:** Сохраняйте текстовые заметки и напоминания отдельно для каждого тайтла.

## 🛠️ Установка из исходного кода (режим разработчика)

Если вы хотите установить расширение вручную из репозитория:

### Для Google Chrome / Microsoft Edge / Яндекс.Браузер:
1. Скачайте репозиторий и распакуйте его (или склонируйте с помощью `git clone`).
2. Откройте браузер и перейдите по адресу `chrome://extensions/`.
3. Включите **«Режим разработчика»** (Developer mode) в правом верхнем углу.
4. Нажмите кнопку **«Загрузить распакованное расширение»** (Load unpacked) в левом верхнем углу.
5. Выберите корневую папку проекта (где находится файл `manifest.json`).

### Для Mozilla Firefox:
1. Перейдите по адресу `about:debugging#/runtime/this-firefox`.
2. Нажмите кнопку **«Загрузить временное дополнение...»** (Load Temporary Add-on...).
3. Выберите файл `manifest.firefox.json` (в режиме разработки) или укажите собранный ZIP-архив из папки релизов.

## 💻 Разработка и тестирование

Проект написан на чистом JavaScript (Vanilla JS) и CSS с использованием современных стандартов Manifest V3.

### Установка зависимостей разработчика:
```bash
npm install
```

### Запуск тестов:
* **Юнит-тесты (Vitest):**
  ```bash
  npm run test:unit
  ```
* **E2E-тесты (Playwright):**
  ```bash
  npm run test:e2e
  ```
* **Все тесты сразу:**
  ```bash
  npm run test
  ```

### Сборка релизных архивов:
Скрипт сборки удаляет тестовый и отладочный код (`DEV_ONLY`) и упаковывает расширение в ZIP-архив:
* **Сборка для Chrome (и Chromium браузеров):**
  ```bash
  npm run build
  ```
  *Результат:* файл `boosty-bookmark-release.zip` в корневой директории.
* **Сборка для Firefox:**
  ```bash
  npm run build:firefox
  ```
  *Результат:* файл `boosty-bookmark-firefox-release.zip` в корневой директории.

## 🔒 Конфиденциальность данных

Все ваши персональные данные (прогресс чтения, списки тайтлов, заметки) хранятся исключительно локально в браузере (через `chrome.storage.local`) или на вашем личном WebDAV-сервере в зашифрованном/архивном виде. Расширение не отправляет данные на сторонние серверы аналитики или сбора информации.
